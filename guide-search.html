검색 방법

이 문서는 Auto QA의 검색 방법 절차를 안내합니다. 아래 단계와 스크린샷을 참고해 따라 해보세요.

1. 검색 조건 설정

조회하려는 콜의 조건을 선택합니다. 채널, 센터, 그룹, 팀, 상담사 명/ID, 고객 CTN, 평가 상태 등을 필요에 맞게 지정하세요.

  • 조직 필터(채널/센터/그룹/팀)의 기본값은 본인 소속 팀으로 설정되어 있습니다.
  • 상담사와 팀장은 본인 소속 팀의 평가결과만 조회 가능합니다.
  • 코치는 본인 소속 센터 내의 모든 팀의 평가결과를 조회 가능합니다.

2. 기간 및 시간 지정

  • 상담 일자(범위)와 상담·통화 시간을 입력하면, 해당 기간에 해당하는 콜만 조회됩니다.
  • 상담 일자는 최대 31일까지 설정 가능합니다.
  • 상담 시간은 콜 인입시간 기준입니다.
  • 상담일자와 상담시간이 동시에 입력된 경우, AND 조건으로 작동합니다.
    예: 2025년 8월 12일 ~ 19일, 09시~19시 → 해당 기간 내 09시~19시 사이의 콜만 조회

3. 검색 실행

모든 조건을 입력한 후 검색 버튼을 클릭하면 결과가 표시됩니다. 조건을 다시 설정하려면 초기화 버튼을 클릭하세요.

4. 검색 결과 확인

목록에서 Conv.ID, 상담 일자/시간, 고객번호, 조직 정보, 상담사, 평가 상태, 점수 등을 확인할 수 있습니다.

5. 엑셀 내려받기

필요 시 엑셀 내려받기 기능으로 결과를 저장·활용하세요.
